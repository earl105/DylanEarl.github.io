Natural Number Calculator

Made with:

The Natural Number Calculator project aims to develop a Java application using the Swing framework and the model-view-controller (MVC) design pattern. This application, unlike typical calculators, performs arithmetic operations with natural numbers that have an extremely large upper limit, and cannot be negative. The project begins by importing a predefined project setup into Eclipse and requires modifications to the model, view, and controller components based on provided interfaces and documentation. Although potential issues like slow performance and stack overflow are noted, the calculator is designed to be robust against user-induced crashes.

Glossary Generator

Made with:

I've generated a batch of HTML files from an input file containing terms and their definitions. The main deliverable is a set of interconnected HTML pages: a top-level index page listing all glossary terms in alphabetical order and individual pages for each term with its definition. Terms in definitions linked to other glossary pages are clickable. The index is named "index.html," and each term page is formatted as "term.html." I specified the input file and output folder paths. The project emphasized software quality factors such as understandability, maintainability, and adherence to coding standards, challenging me to deliver a realistic, component-based software product without predefined templates or frameworks.
